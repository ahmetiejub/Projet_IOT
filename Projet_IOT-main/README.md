Ce GitHub est créé pour notre projet IoT. Chaque groupe possède sa propre branche dédiée pour développer sa partie.

Voici la marche à suivre :

Cloner le dépôt complet sur votre ordinateur.

Basculer (Checkout) immédiatement sur la branche de votre groupe (ex: dev-client-iot) avant de commencer à coder.

Travailler dans votre répertoire local.

Push vos modifications régulièrement sur votre branche spécifique.

Attention : Ne faites jamais de push directement sur la branche main. On fusionnera (Merge) tout ensemble à la fin quand chaque partie sera fonctionnelle."
